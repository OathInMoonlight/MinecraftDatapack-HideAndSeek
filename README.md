# MinecraftDatapack-HideAndSeek
这是一个用于原版服务器的 **方块躲猫猫** 小游戏数据包，基于 **1.20.4** 版本开发

## 使用方法
[点击此处下载数据包](https://github.com/OathInMoonlight/MinecraftDatapack-HideAndSeek/releases)  
将下载好的数据包放置在你游戏存档的`存档名/datapacks/`目录下  
打开游戏进入存档，使用`/reload`命令加载数据包  
按照提示设置初始参数即可开始游戏  

## 基本规则
游戏开始将会随机分配玩家为 **躲藏者** 和 **搜查者**
### 躲藏者
1. **躲藏者** 会被分配一个方块，原地不动5秒即可变为这个方块
2. **躲藏者** 被攻击时会变回实体
3. **躲藏者** 可以释放技能 **嘲讽** 来赚取 **挑衅点数**
4. **嘲讽** 会在 **躲藏者** 的头顶释放烟花来暴露位置
5. **挑衅点数** 可以随机换取 **瞬间治疗药水** 或者 **隐身药水**
6. **躲藏者** 在倒计时结束前未被击杀则获胜
7. **躲藏者** 若被击杀则变更至 **搜查者** 阵营
### 搜查者
1. **搜查者** 在游戏开始需等待一定时间后进入游戏区域
2. **搜查者** 会被分配一个 **罗盘** 指向最近的 **躲藏者** 的模糊位置
3. 随着 **躲藏者** 被击杀，**罗盘** 的指向会愈发不精确
4. 若 **搜查者** 一定时间未击杀 **躲藏者**，一位随机的 **搜查者** 将获得一个 **圣光** 技能
5. **搜查者** 释放 **圣光** 可以使其最近的 **躲藏者** 显形
6. **搜查者** 若死亡需等待一定时间后复活
7. **搜查者** 在规定时间内击杀所有 **躲藏者** 则获胜
### 记分板
1. **躲藏者** 若获得游戏胜利则获得分数
2. **躲藏者** 若获得游戏胜利，其累计的 **挑衅点数** 也会转化为分数
3. **搜查者** 需找出方块状态下的 **躲藏者** 以获得分数

## 可自定义项
使用函数`set_game_spawn_point`设置游戏内出生点  
使用函数`set_hider_cooling_time`设置躲藏者嘲讽技能冷却时间  
使用函数`set_hider_reward_point`设置多少嘲讽点数兑换一次药水  
使用函数`set_lobby`设置地图大厅位置  
使用函数`set_respawn_time`设置重生等待时间  
使用函数`set_seek_time`设置搜查者搜查时间  
使用函数`set_seeker_compas_range_increment`设置搜查者罗盘范围随躲藏者数量减少的增量  
使用函数`set_seeker_compass_range`设置搜查者罗盘初始范围  
使用函数`set_seeker_num`设置初始搜查者数量  
使用函数`set_seeker_skill_interval_time`设置搜查者圣光发放间隔  
使用函数`set_seeker_wating_point`设置搜查者等待位置  
使用函数`set_seeker_wating_time`设置搜查者初始等待时间
